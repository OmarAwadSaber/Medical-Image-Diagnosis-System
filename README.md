# Medical Image Diagnosis (Supervised Learning)

This repository organizes the project into phases and keeps large artifacts out of Git.

## Structure

- data/ (dataset links only; no raw data committed)
- phase1/phase1_classical_ml.ipynb
- phase2/phase2_deep_learning.ipynb
- phase3/ (reserved for sequence models and attention work)
- models/ (links to saved models)
- report/ (final report)

## Workflow (GitHub + Colab)

1. Open a notebook in Colab.
2. File > Save a copy in GitHub.
3. Work on your own branch and merge via pull request.

See data/README.md and models/README.md for links.
